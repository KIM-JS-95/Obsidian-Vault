```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        ans = nums[0]
        csum = 0

        for i in nums:
            csum +=i
            if csum > ans:
                ans = csum
            if csum < 0:
                csum = 0
        return ans
```
